## üíæ Installation

Just use `pip`!

```bash
conda create -n py3.9-ggce python=3.9 -y
conda activate py3.9-ggce
pip install ggce
```

We provide detailed information on _advanced_ installation at our [documentation](https://matthewcarbone.github.io/GGCE/installation.html).
